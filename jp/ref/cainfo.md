# cainfo
```
tip> cainfo
Usage: cainfo EPICS_record
Show information of the EPICS_record.
```

example
```
tip> cainfo monitor:ai1
monitor:ai1
    State:         connected
    Host:          some.host.name
    Access:        read=1, write=1
    Data type:     DBR_DOUBLE (native: DBF_DOUBLE)
    Element count: 1
```
