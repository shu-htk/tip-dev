# leg
```
tip> leg
Usage: leg show [pos] [(opt)]
     : leg ls
     : leg clear
Draw legend on the existing graph (2nd arg is show).
pos:
 first char:
 R : Rgiht (default)
 L : Left
 M : Middle
 second char:
  T: Top (default)
  B: Bottom
  M: Middle
Other commands at 2nd arg:
 ls:    list legend strings.
 clear: clear legend strings.
```
