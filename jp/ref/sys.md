# sys
```
tip> sys
usage: sys [command]
 execute system command
```

`sys` execute shell command in the tip interpreter

example
```
tip> sys ps
    PID TTY          TIME CMD
    335 pts/0    00:00:00 bash
   6503 pts/0    00:00:00 emacs
   6518 pts/0    00:00:00 tip
   6519 pts/0    00:00:00 sh
   6520 pts/0    00:00:00 ps
```
