# text
```
tip> text
Usage: text x y "text" [(opt)]
Escape sequences:
  #u  superscript (end with #d)
  #d  subscript (end with #u)
  #g  next character becomes Greek
  #fn normal (sans-serif) font
  #fr Roman (serif) font
  #fi Italic font
```

## option

- `tc:` text color, (default is black)
- `tp:` text pos(0:L to 1:R), (default is 0)
- `tr:` text rotation(deg), (default is 0)
- `ts:` text size, (default is 1)
