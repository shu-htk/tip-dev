# box
```
usage: box x0 x1 y0 y1 [(opt)]
```

available options

|option| description|
|-|-|
|(tl:"title")|same as `title` command|
|(xl:"x-label")|same as `xlab` command|
|(yl:"y-label")|same as `ylab` command|
|(bo:N)|box drawing order 0:hor, 1:ver|
|(bp:N)|box position of divided area|

see [ex16: plot divided box by vertical order](../ex/ex16.md)
